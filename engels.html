Completed Projects

Museum

This was a collaborative project where we worked with game development students and design students to create a unique piece for the Amsterdam Museum. The project was inspired by an artwork titled "OkkernotenBoom,". My role in this project was to design and develop an informative website dedicated to the artwork. The website provided visitors with detailed information about the background, artistic process, and cultural significance of the "OkkernotenBoom." By offering an accessible and engaging online platform, the website allowed users to explore the deeper meanings and history behind the artwork.

Webshop

This project was a solo venture where I developed an online game store, similar to platforms like Steam or Epic Games. The website includes several key features, such as a search function, filtering options, and the ability to add products to a shopping cart. To achieve this, I used JavaScript and JSON, which allowed me to create dynamic and interactive functionalities. The development process was technically challenging, as it required a deep understanding of user experience and efficient coding. This project significantly enhanced my skills in web development, particularly in handling data and improving the user interface.

Tulpreizen

This project provided me with the opportunity to apply my knowledge of Sass in a practical setting for the first time. The website was intended as a guide to popular tourist attractions in Amsterdam, offering detailed information about various places to visit. While the project’s primary goal was to create a useful platform for travelers, my personal objective was to practice and refine my use of Sass, enabling more efficient and modular CSS styling. This hands-on experience not only helped me better understand the power of Sass but also improved the overall structure and design of the website, contributing to my growth as a web developer.

Fontshowcase

This project focused on developing a dynamic website where I once again utilized Sass for a more structured and efficient CSS. I also implemented JavaScript to allow the fonts to change based on what was typed into the input field. Additionally, I added both dark mode and white mode for an improved user experience. During this project, I further developed my animation skills, making the interactions smoother and more visually appealing. Combining Sass, JavaScript, and animation gave me greater control over both the design and functionality of the website.

Duurzaamhuis

This project was aimed at putting my knowledge of Arduino into practice. The goal was to use the Arduino to collect data, such as temperature, and send that information to the website. Although I had learned a lot about Arduino in theory, this was the first time I actually applied it in a project. The project greatly helped me understand the technical connection between hardware and software and how to apply it in various scenarios. It was a valuable hands-on experience that deepened my understanding of real-time data collection.

Pokemon

In this project, I worked with a Pokémon API to display random Pokémon and their types. The goal was to dynamically generate content so that different Pokémon and their attributes would appear each time on the website. I also used CSS grids to create graphs, which helped improve my knowledge of layout structures. This project combined API integration with visual representation, making it a great opportunity to refine my skills in front-end development and data visualization. The project was educational and fun, as it allowed me to interact with the Pokémon universe.
