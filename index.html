Afgeronde Projecten

Museum

Dit was een project waarin we samenwerkten met studenten game-ontwikkeling en studenten ruimtelijke vormgeving om gezamenlijk een kunstwerk te creëren voor het Amsterdam Museum. Het project was gebaseerd op een kunstwerk genaamd "OkkernotenBoom,". Ik had de werk om een informatieve website te ontwikkelen die niet alleen uitleg gaf over het kunstwerk, maar ook de geschiedenis, de betekenis en de artistieke intenties achter het werk belichtte. Op deze manier konden bezoekers dieper inzicht krijgen in de artistieke als culturele context.

Webshop

Dit project was een solo-onderneming waarbij ik een webshop heb ontwikkeld voor de verkoop van online games, vergelijkbaar met Steam of Epic Games. De website bevat verschillende functies, waaronder zoeken, filteren, en de mogelijkheid om producten aan een winkelwagen toe te voegen. Voor de ontwikkeling van deze functies heb ik veel gebruik gemaakt van JavaScript en JSON. Deze programmeertalen hielpen bij het realiseren van een dynamische en gebruiksvriendelijke ervaring voor de gebruiker, wat het project zowel technisch uitdagend als lonend maakte.

Tulpreizen

Dit project gaf mij de kans om voor het eerst mijn kennis van Sass in de praktijk te brengen. De website was bedoeld als een gids voor toeristische bezienswaardigheden in Amsterdam, met informatie over populaire plekken om te bezoeken. Hoewel het oorspronkelijke doel was om een reisplatform te creëren, lag mijn echte interesse in het leren gebruiken van Sass om efficiënter te werken met CSS. Dit project gaf me de mogelijkheid om te experimenteren met styling en structuur, waardoor mijn vaardigheden verder werden aangescherpt.

Fontshowcase

Dit project was gericht op het ontwikkelen van een dynamische website waarbij ik gebruik maakte van Sass voor een meer gestructureerde en efficiënte CSS. Ik implementeerde ook JavaScript om de lettertypen te laten veranderen op basis van wat er in de invoerbalk werd geschreven. Daarnaast heb ik zowel een dark mode als een white mode toegevoegd. Tijdens dit project heb ik ook mijn vaardigheden in animaties verder ontwikkeld, waardoor de interacties vloeiender en visueel aantrekkelijker werden.

Duurzaamhuis

Dit project was bedoeld om mijn kennis van Arduino in de praktijk te brengen. Het doel was om de Arduino te gebruiken om gegevens, zoals temperatuur, te verzamelen en deze informatie door te sturen naar de website. Hoewel ik veel had geleerd over Arduino in theorie, was dit de eerste keer dat ik het daadwerkelijk in een project toepaste. Het project heeft me enorm geholpen om de technische verbinding tussen hardware en software beter te begrijpen en toe te passen in verschillende situaties.

Pokemon

In dit project werkte ik met een Pokémon API om willekeurige Pokémon en hun typen weer te geven. Het doel was om dynamisch gegenereerde inhoud te creëren waarbij elke keer verschillende Pokémon en hun kenmerken op de website verschenen. Ik gebruikte ook CSS-grids om grafieken te maken, wat me hielp mijn kennis van lay-outstructuren te verbeteren. Het was een goede gelegenheid om mijn vaardigheden in front-end ontwikkeling en gegevensvisualisatie verder te verfijnen.
